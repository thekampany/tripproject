Itinerary JSON Schema

Dit is het JSON-schema dat gebruikt wordt bij het importeren van een Itinerary Idea.

{{ schema|safe }}

Toelichting:

  • name: Naam van de itinerary (string, verplicht)
  • days: Array van dagen, elk met:
    • day_sequence: volgorde van de dag (integer, verplicht)
    • day_description: beschrijving van de dag (string, optioneel)
    • day_possible_date: mogelijke datum (YYYY-MM-DD, optioneel)
    • day_locations: array van locaties met sequence, lat, long, radius, description
    • overnightlocation: object met lat, long, radius, description
